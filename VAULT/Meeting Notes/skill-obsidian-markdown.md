# Skill — Obsidian Markdown

## Overview
הקובץ `SKILLS/obsidian/obsidian-markdown/SKILL.md` מגדיר skill ליצירה ועריכה של Obsidian Flavored Markdown. מכסה את התחביר הייחודי של Obsidian מעבר ל-CommonMark: wikilinks (`[[Note]]`), embeds (`![[embed]]`), callouts (`> [!type]`), frontmatter properties, tags, comments (`%% %%`), highlight (`==text==`), LaTeX, ו-Mermaid diagrams. כולל workflow מ-frontmatter עד verification ב-reading view. מפנה לקבצי reference חיצוניים (PROPERTIES.md, EMBEDS.md, CALLOUTS.md) שאינם קיימים עדיין בפרויקט.
