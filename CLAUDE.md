# CLAUDE.md

Этот файл предоставляет инструкции Claude Code при работе с кодом в этом репозитории.

## Проект

COLUMBUS — система из 5 агентов для производства контента (основана на THE 5 AGENTS).

## CEO Agent — обязательно в каждой сессии

**В начале каждой сессии и перед каждой командой** прочитай `.claude/AGENTS/ceo-agent/AGENT.md` и действуй согласно ему:
1. CEO — **всегда** первый активный агент, даже если задача кажется простой
2. Проанализируй intent команды
3. Направь к нужному агенту согласно Routing Logic в AGENT.md
4. Объяви о маршрутизации перед выполнением: "Передаю [агенту] — [причина]"
5. Если непонятно — задай один уточняющий вопрос перед передачей

## Vault Workflow — обязательно в каждой сессии

**В начале каждой сессии и перед каждой командой** следуй Phase 1 из `SKILLS/obsidian/obsidian-vault-workflow/SKILL.md`:
1. Определи тему задачи
2. Найди соответствующий topic file в `VAULT/Meeting Notes/` и прочитай его
3. Прочитай 2–3 последних записи в `VAULT/Meeting Notes/_index.md`
4. Сообщи в одном предложении, что загрузил перед началом работы

**В конце каждой сессии** следуй Phase 2:
1. Создай или обнови topic file в `VAULT/Meeting Notes/`
2. Добавь датированную запись сессии со статус-тегом
3. Обнови `_index.md` если создан новый файл
4. Перечитай файл для проверки перед завершением
