# Навык: Веб-поиск — Метод качественного поиска Хен

## Когда использовать
Когда Хен получает запрос на поиск от Реувена и переходит к шагу 3 (веб-поиск) своего Workflow.

---

## Шаг 1 — Построение сфокусированного запроса

**Принцип:** Широкий запрос возвращает мусор. Сфокусированный запрос возвращает золото.

| Тип запроса | Пример хорошего запроса | Пример плохого запроса |
|-------------|------------------------|----------------------|
| Профессиональная статья | `"content marketing B2B 2024 case study"` | `"маркетинг"` |
| Новости | `"OpenAI GPT-5 release site:techcrunch.com OR site:theverge.com"` | `"новости AI"` |
| Академический | `"LLM hallucination reduction methods filetype:pdf"` | `"проблемы AI"` |
| Русский | `"контент-маркетинг для малого бизнеса 2024"` | `"маркетинг"` |

**Тактики:**
- Добавляй год (`2024`, `2025`) для актуальных результатов
- Используй `site:` для конкретных сайтов
- Добавляй `"case study"` / `"guide"` / `"how to"` для глубокого контента
- Пробуй и на русском, и на английском, если тема двуязычная

---

## Шаг 2 — Оценка результата

Для каждого результата WebSearch проверь:

| Критерий | Проверка |
|----------|---------|
| Автор | Есть ли чёткое имя автора? |
| Дата | Актуальна ли статья? (не старше 2 лет, если не требуется иное) |
| Сайт | Известен ли и надёжен ли сайт? |
| Объём | Есть ли реальный контент (не только заголовок)? |
| Ссылки | Ссылается ли статья на источники? (признак качества) |

**Если результат проходит менее 3 критериев — искать заново с другим запросом.**

---

## Шаг 3 — Структурированное извлечение

После нахождения качественного источника, извлеки по схеме:

```
Заголовок: [название статьи]
URL: [полная ссылка]
Автор: [имя автора] / [название сайта]
Дата: [YYYY-MM-DD]

Резюме (3–5 предложений):
[...]

Ключевые цитаты:
1. "[цитата]" — [источник]
2. "[цитата]" — [источник]

Ключевые моменты:
- [...]
- [...]
- [...]
```

---

## Советы по поиску на русскоязычные темы

- Ищи и на русском, и на английском — иногда на английском лучше источники
- Предпочтительные сайты (RU): vc.ru, habr.com, rbc.ru, forbes.ru, tadviser.ru
- Для международного контента: techcrunch.com, hubspot.com, contentmarketinginstitute.com
- Для B2B-контента: ищи в профессиональных форумах и корпоративных блогах

---

## Частые сбои

| Сбой | Решение |
|------|---------|
| Запрос возвращает рекламу | Добавь `-ad -sponsored` к запросу |
| Результаты слишком старые | Добавь `after:2023` к запросу |
| Только поверхностные результаты | Переключись на английский с переводом темы |
| WebSearch возвращает только короткий текст | Выполни WebFetch по конкретному URL для получения полного контента |
