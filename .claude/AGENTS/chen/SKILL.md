# Skill: Web Search — חיפוש רשת איכותי

## מתי להשתמש
כשחן מקבלת בקשת חיפוש מראובן ועוברת לשלב 3 (חיפוש ברשת) ב-Workflow שלה.

---

## שלב 1 — בניית Query ממוקד

**עקרון:** query רחב מחזיר זבל. query ממוקד מחזיר זהב.

| סוג בקשה | דוגמת query טובה | דוגמת query גרועה |
|----------|-----------------|------------------|
| מאמר מקצועי | `"content marketing B2B 2024 case study"` | `"שיווק תוכן"` |
| חדשות | `"OpenAI GPT-5 release site:techcrunch.com OR site:theverge.com"` | `"חדשות AI"` |
| אקדמי | `"LLM hallucination reduction methods filetype:pdf"` | `"בעיות AI"` |
| עברית | `"שיווק תוכן לעסקים קטנים 2024"` | `"שיווק"` |

**טקטיקות:**
- הוסף שנה (`2024`, `2025`) לתוצאות עדכניות
- השתמש ב-`site:` לאתרים ספציפיים
- הוסף `"case study"` / `"guide"` / `"how to"` לתוכן מעמיק
- נסה גם בעברית וגם באנגלית אם הנושא דו-לשוני

---

## שלב 2 — הערכת תוצאה

לכל תוצאה שמחזיר WebSearch, בדוק:

| קריטריון | בדיקה |
|----------|-------|
| מחבר | האם יש שם מחבר ברור? |
| תאריך | האם המאמר עדכני? (לא יותר מ-2 שנים אחורה, אלא אם נדרש) |
| אתר | האם האתר מוכר ואמין? |
| אורך | האם יש תוכן ממשי (לא רק כותרת)? |
| קישורים | האם המאמר מקשר למקורות? (סימן לאיכות) |

**אם תוצאה עוברת פחות מ-3 קריטריונים — חפש שוב עם query שונה.**

## שלב 3 — חילוץ מובנה

לאחר שנמצא מקור איכותי, חלץ לפי הסכמה:

```
כותרת: [כותרת המאמר]
URL: [קישור מלא]
מחבר: [שם מחבר] / [שם אתר]
תאריך: [YYYY-MM-DD]

תקציר (3–5 משפטים):
[...]

ציטוטים מרכזיים:
1. "[ציטוט]" — [מקור]
2. "[ציטוט]" — [מקור]

נקודות מפתח:
- [...]
- [...]
- [...]
```

- חפש גם בעברית וגם באנגלית — לעיתים יש מקורות טובים יותר באנגלית

## כשלים נפוצים

| כשל | פתרון |
|-----|-------|
| query מחזיר פרסומות | הוסף `-ad -sponsored` לquery |
| תוצאות ישנות מדי | הוסף `after:2023` לquery |
| רק תוצאות עברית שטחיות | עבור לאנגלית עם נושא תרגום |
| WebSearch מחזיר טקסט קצר בלבד | בצע WebFetch לURL הספציפי לקבלת תוכן מלא |
